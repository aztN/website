My Portfolio Website
====================

Description:
------------
This is my personal portfolio website showcasing my front-end development journey, projects, certificates, and resume. 
It includes different sections like Home, About Me, Projects, Certificates, Contact, and Resume.

Technologies Used:
------------------
- HTML5
- CSS3
- JavaScript (Vanilla)
- FreeCodeCamp Projects
- Google Fonts (Poppins)

How to Use:
-----------
1. Clone or download this repository.
2. Open the `index.html` file in a web browser.
3. Make sure images and assets are correctly linked in the project folder.
4. You can update your personal information, images, and links to reflect your own projects.
